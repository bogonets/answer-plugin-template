# Frontend web for answer-plugin-template

## Project setup
```.shell
yarn install
```

### Compiles and hot-reloads for development
```.shell
yarn serve
```

### Compiles and minifies for production
```.shell
yarn build
```

### Run your unit tests
```.shell
yarn test:unit
```

### Run your end-to-end tests
```.shell
yarn test:e2e
```

### Lints and fixes files
```.shell
yarn lint
```

### Link @recc/api package
```.shell
yarn link:api
```
