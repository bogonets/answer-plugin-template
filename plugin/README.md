# answer-plugin-template

Python plugin for the ANSWER backend.

## Script usage

### Formatting
```
bash black.sh
```

### PEP8
```
bash flake8.sh
```

### Import ordering
```
bash isort.sh
```

### Type analysis
```
bash mypy.sh
```

### Testing
```
bash pytest.sh
```
